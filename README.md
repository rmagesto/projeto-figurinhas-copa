# Projeto Figurinhas da Copa

Este projeto implementa um sistema de gerenciamento de figurinhas da Copa do Mundo usando estruturas de dados encadeadas em Python.

## Descrição

O sistema permite que usuários colecionem, organizem e troquem figurinhas. As principais funcionalidades incluem:

- Inserir figurinhas no álbum
- Remover figurinhas do álbum
- Consultar figurinhas por número
- Listar o álbum completo
- Calcular a porcentagem de conclusão do álbum
- Armazenar e exibir figurinhas repetidas
- Buscar figurinhas por jogador ou seleção
- Propor trocas de figurinhas entre usuários
- Efetuar trocas automáticas de repetidas
- Salvar e carregar dados em formatos JSON, CSV e TXT

## Estrutura do projeto

- `main.py`: ponto de entrada com interface de linha de comando.
- `figurinhas.py`: implementação das classes e estruturas de dados do projeto.

## Como executar

Certifique-se de ter Python 3 instalado.

1. Abra um terminal na pasta do projeto.
2. Execute:

```bash
python3 main.py
```

A aplicação iniciará com um menu interativo para manipular o álbum de figurinhas.

## Requisitos

- Python 3

## Uso básico

O menu principal oferece opções para:

- Adicionar figurinhas ao álbum
- Ver as figurinhas repetidas
- Buscar figurinhas por jogador ou seleção
- Salvar/Carregar o estado do usuário em JSON
- Exportar dados para CSV ou TXT
- Visualizar o histórico de trocas
